# Changelog

## v0.2.0 - 2026-06-28

- Added session undo for transferred RIGs, copied IRs, and generated or updated setlists.
- Updated setlist handling so imported rigs can be appended to an existing setlist with the same name.
- Normalized setlist names to uppercase in the transfer dialog and on save.
- Improved browser analysis panels to show short preset and IR names instead of long relative paths.
- Added `HeadRush MX5:` device action in the footer for connected-device eject and sync workflow.
- Refreshed README screenshots and workflow documentation.

## v0.1.0 - 2026-06-27

- First public release.
